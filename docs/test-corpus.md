# Тестовый корпус для ручной проверки

Все данные **синтетические** (контрольные суммы вычислены, реальные реквизиты не
используются). Документ проверялся прогоном через движок: **все перечисленные
сущности маскируются, все «ловушки» остаются нетронутыми** (кроме случаев,
отмеченных ниже).

## Как пользоваться

1. Скопируйте блок ниже целиком, вставьте в поле ввода AI-чата.
2. Нажмите «Замаскировать» (или ⌥⇧M / Alt+Shift+M).
3. Сверьте с разделами «Ожидается» и «Ловушки».
4. Для даты рождения включите тип **DATE_OF_BIRTH** в настройках (по умолчанию выкл.).
