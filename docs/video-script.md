# Сценарий демо-видео (~75 секунд)

Все данные в кадре — **синтетические**. Экран: браузер с открытым ChatGPT,
расширение установлено.

| Время | Кадр | Действие / закадр |
|---|---|---|
| 0:00–0:08 | Логотип + заголовок | «PII Firewall — локальный файрвол персональных данных. Всё работает внутри браузера.» |
| 0:08–0:18 | `chrome://extensions`, карточка расширения, клик по «Подробнее» → Разрешения | «Смотрите: разрешения — только локальное хранилище. Ни одного сетевого. Данные физически не уходят с устройства.» |
| 0:18–0:30 | ChatGPT, вставляем синтетический договор с ФИО, ИНН, телефоном | «Пишем реальный рабочий промпт — договор с персональными данными клиента.» |
| 0:30–0:40 | Клик «Замаскировать» в виджете. Текст меняется на `[PERSON_1]`, `[RU_INN_1]`… Чипы сущностей | «Один клик — и все ПДн заменены на обратимые метки. Видно, что именно найдено.» |
| 0:40–0:50 | Отправляем. Показать: перед отправкой без маскировки — модал Guard | «Забыли замаскировать? Guard-режим перехватит отправку и предупредит.» |
| 0:50–1:02 | Ответ ChatGPT с метками. Выделяем, Alt+Shift+U → оригиналы | «AI отвечает, сохраняя метки. Восстановление — и оригиналы возвращаются локально, прямо в браузере.» |
| 1:02–1:10 | Popup: счётчик «предотвращено утечек», кнопка «Забыть всё» | «Счётчик предотвращённых утечек. Хранилище очищается при закрытии браузера.» |
| 1:10–1:15 | Логотип + «Free. Ноль сетевых разрешений.» | Финальный кадр с призывом. |

## Заметки по записи
- Разрешение экрана 1280×800 или 1920×1080.
- Показать `manifest.json` `"permissions": ["storage"]` крупным планом — ключевой момент доверия.
- Не использовать реальные ПДн ни в одном кадре.
- Скриншоты для листинга снять из этих же сцен (кадры 0:08, 0:30, 0:40, 0:50, 1:02).
