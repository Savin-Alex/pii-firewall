# Листинг для маркетплейса (RU)

## Название
PII Firewall — локальный файрвол персональных данных

## Короткое описание (до 132 символов)
Маскирует ПДн прямо в браузере до отправки в AI-чат и восстанавливает ответ. Ноль сетевых разрешений — проверьте manifest.

## Полное описание

**PII Firewall** обнаруживает и обратимо маскирует персональные данные
**полностью внутри браузера**, прежде чем ваш промпт уйдёт в ChatGPT, Claude,
Gemini, Perplexity или Poe — и восстанавливает оригиналы в ответе локально.

🔒 **Ноль сетевых разрешений.** Расширение не запрашивает доступ к сети.
Разрешения — `storage`, `activeTab`, `contextMenus` — ни одно не сетевое и не
даёт `host_permissions`. Проверьте сами в `manifest.json`. Ваши данные физически
не могут покинуть
устройство через это расширение.

**Что распознаётся (с проверкой контрольных сумм):**
- ФИО (эвристика для русских имён и отчеств)
- Email, телефоны (РФ и международные)
- Банковские карты (Луна), IBAN (mod-97)
- ИНН, СНИЛС, ОГРН, паспорт РФ, полис ОМС — с проверкой контрольных сумм
- SSN (США), IP-адреса
- Секреты: API-ключи, токены, пароли

**Как это работает:**
1. Пишете промпт с персональными данными.
2. Нажимаете «Замаскировать» (или Alt+Shift+M) — данные заменяются на метки
   `[PERSON_1]`, `[RU_SNILS_1]`…
3. Отправляете в AI. Guard-режим предупредит, если вы забыли замаскировать.
4. Ответ приходит с метками — «Восстановить» (Alt+Shift+U) вернёт оригиналы
   локально.

**Приватность по умолчанию:** хранилище очищается при закрытии браузера.
Опционально — шифрованное постоянное хранилище (AES-GCM, ключ из вашего
пароля).

### Честные ограничения
- Распознавание имён — **эвристика**, не нейросетевой NER: возможны пропуски
  (редкие имена, сложные падежи) и ложные срабатывания. Всегда проверяйте
  подсветку перед отправкой.
- Инструмент **снижения риска** утечки ПДн, **не сертифицированное средство
  защиты информации**.
- Данные в примерах и тестах — синтетические.

### Модель без сервера: офлайн-лицензия
Ключевой принцип продукта — **ноль сети** — исключает серверные подписки и
телеметрию. Поэтому Pro/Team разблокируются **офлайн-лицензионным ключом**:
подписанный ключ покупается на маркетплейсе и проверяется **локально**
(криптоподпись, без обращения к серверу). Privacy-обещание сохраняется полностью:
даже платная версия ничего не отправляет в сеть.

### B2B под 152-ФЗ (главный коммерческий угол)
Компаниям — снижение риска утечки ПДн сотрудников и клиентов в ChatGPT, GigaChat,
YandexGPT по всей команде. Это ответ на реальную боль комплаенса (152-ФЗ / GDPR)
и основа номинации «Проект с максимальным коммерческим потенциалом».

**Уже в продукте (бесплатно) — маскирование корпоративных реквизитов РФ:**
ИНН/ОГРН юрлиц, **расчётный и корр. счёт с проверкой контрольного ключа по БИК**,
**БИК**, **КПП** — то, что утекает в промпты «помоги составить договор/платёжку».

**Планируемые Team/Business-функции:**
- **Командная политика** — админ задаёт один конфиг (какие типы/сайты маскировать,
  guard принудительно вкл.) и раздаёт сотрудникам; локальный «залоченный» режим,
  который пользователь не может отключить на критичных сайтах.
- **Локальный аудит-лог** — журнал «предотвращено N утечек, какие типы, когда»
  (только маскированно, без сырых значений) как доказательство для комплаенса.
- **Свои словари и правила** — маскирование внутренних сущностей компании
  (названия проектов, внутренние ID, коды контрагентов).
- **Опциональный self-hosted анонимайзер** — для команд, которым нужен серверный
  NER и централизованный аудит (off по умолчанию, ставится внутри периметра).
- **Пакетная лицензия** — офлайн-ключи на N сотрудников, без обращения к серверу.
