# Соответствие правилам хакатона Kodik Launchpad

Документ фиксирует соответствие проекта требованиям правил
(hackathon.kodik.ru/rules). Данные-примеры — синтетические.

## Гарантии §8.2.6

Программа (расширение PII Firewall) гарантирует, что она:

| Требование §8.2.6 | Как выполнено |
|---|---|
| Нет порнографии и материалов сексуального характера | В проекте отсутствуют. |
| Нет насилия, разжигания ненависти, вражды | Отсутствуют. |
| Нет экстремистских материалов | Отсутствуют. |
| **Нет персональных данных третьих лиц** | Все значения в тестах, корпусе, онбординге и демо — **синтетические**: сгенерированы, контрольные суммы (ИНН/СНИЛС/ОГРН/карты/ОМС/IBAN) вычислены алгоритмически. Реальные реквизиты не используются. Генератор: `scripts/generate-corpus.ts`. |
| Нет чужих изображений и объектов интеллектуальной собственности | Иконки собраны из `PII Logo.png` (ассет проекта) генератором `scripts/gen-icons.mjs`. Сторонних изображений нет. |
| Нет нарушения авторских прав | Собственный код. Зависимости — только dev-time, лицензия MIT (vite, vitest, typescript, jsdom); в поставляемое расширение не включаются. |
| **Нет вредоносного/шпионского ПО и трекинга** | **Ноль сетевых разрешений**: `manifest.json` запрашивает `storage`, `activeTab`, `contextMenus` — ни одно не даёт доступа к сети, нет `host_permissions`. Нет fetch/XHR/WebSocket/beacon в коде, нет аналитики и телеметрии. Проверяемо статически. См. [PRIVACY.md](PRIVACY.md). |

## §5.7.1 — создано в дни Хакатона

Проект создан в новом репозитории; все коммиты датированы **02–04.07.2026**,
внутри окна хакатона (см. [CHANGELOG.md](CHANGELOG.md) и историю git).
Движок детекции написан с нуля на TypeScript по публичным алгоритмам
(Luhn, контрольные суммы ИНН/СНИЛС/ОГРН, IBAN mod-97); сторонний код не
копировался.

## §5.7 — заявленные функции

Расширение выполняет заявленное: обнаружение ПДн в браузере, обратимое
маскирование до отправки в AI-чат, восстановление ответа локально, guard-режим.
Автоматические тесты (`npm test`) и загружаемая сборка (`npm run build`)
подтверждают работоспособность; ручной E2E-прогон пройден на ChatGPT, Gemini,
Grok и YandexGPT (Alice) — см. [docs/e2e-checklist.md](docs/e2e-checklist.md).

## Честные ограничения

PII Firewall — **инструмент снижения риска** утечки ПДн, **не сертифицированное
средство защиты информации**. Распознавание имён — эвристика (не нейросетевой
NER), возможны пропуски и ложные срабатывания; метрики приведены честно в
[README.md](README.md).
