# Changelog

Все значимые изменения проекта. Формат — [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/).
Все даты — внутри окна хакатона Kodik Launchpad (создано с нуля, §5.7.1).

## [0.5.0] — 2026-07-04

### Добавлено
- **M4 — Guard-режим:** перехват отправки (Enter в capture-фазе + клик по
  кнопке) → модал «Замаскировать и отправить / Отправить как есть / Отмена» с
  подсветкой ПДн (экранирование, без XSS). Глобальный тумблер + переключатель
  на каждый сайт. Счётчик «предотвращено утечек» в popup.
- **M5 — Popup и настройки:** popup (сайт, счётчик, guard-тумблер, «забыть»,
  демо-режим, недавние сессии с маскированным превью). Options (все 14 типов,
  guard, инструкция, язык, стиль меток, экспорт/импорт настроек, список сайтов,
  опциональное AES-GCM-хранилище). Настройки читаются content-скриптом на лету.
- **i18n:** RU (основная) + EN через `_locales` и `default_locale`; UI popup,
  options, виджета и guard локализованы.
- **M6 — Onboarding:** тур из 3 шагов с синтетическим примером, открывается при
  установке и из popup; скрипт вынесен во внешний файл (CSP). Демо-режим с
  авто-вставкой примера в поле ChatGPT.
- **M8 — Упаковка:** иконки 16/32/48/128 из `PII Logo.png`, store-zip,
  листинг ([docs/listing-ru.md](docs/listing-ru.md)), сценарий видео.
- Стили меток `[X_1]` / `{{X_1}}` / `«X_1»` (restore стиле-агностичен).

## [0.3.0] — 2026-07-03

### Добавлено
- **M2 — Обратимое маскирование и vault:** плейсхолдеры с полным именем типа
  (`[RU_SNILS_1]`), стабильная кореференция, защита от коллизий с вставленными
  метками, изоляция и миграция сессий, опциональная AES-GCM/PBKDF2
  персистентность. Property-тест round-trip на 120 синтетических документах.
- **M3 — Content script:** плавающий виджет (Shadow DOM, тёмная тема), устойчивый
  ввод (нативный сеттер + `InputEvent`; `selectAll+insertText` для ProseMirror),
  конфиг сайтов (ChatGPT, Claude, Gemini, Perplexity, Poe), горячие клавиши через
  manifest `commands`, сборка загружаемого расширения.

### Исправлено
- Стабильные переносы строк в contenteditable (поблочное чтение).
- Удалены AKIA-подобные синтетические ключи из корпуса (GitHub Push Protection).

## [0.1.0] — 2026-07-02

### Добавлено
- **M1 — Движок детекции** (чистый TypeScript, без DOM, ноль runtime-зависимостей):
  нормализация NFKC + фолдинг гомоглифов; детекторы EMAIL, PHONE, CARD (Luhn),
  IBAN (mod-97), RU_SNILS/RU_INN/RU_OGRN/RU_PASSPORT/RU_OMS с контрольными
  суммами, US_SSN, IP_ADDRESS, SECRET (JWT/AWS/kv/энтропия), PERSON (RU/EN
  эвристика), DATE_OF_BIRTH (opt-in).
- Синтетический корпус (93 фразы, негативы) с CI-порогами P≥0.98 / R≥0.95 /
  persons R≥0.65; юнит-тесты контрольных сумм.
